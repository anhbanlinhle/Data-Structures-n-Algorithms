Function sum:
  sum = 0;
  for ( i = 0; i < n; i + +)              O(n)
    for ( j = i + 1; j < = n; j + +)      O(n)
      for ( k = 1; k < 10; k + +)         O(n)
        sum = sum + i * j * k ;

=> O(n^3)
